# UI/UX Guidelines

## Design System
- **Typography**: Inter (Google Font) – 400/500/600 weights.
- **Color Palette**: Primary HSL(210, 70%, 55%); Secondary HSL(340, 65%, 50%); Neutral greys for backgrounds and text.
- **Spacing**: 8px baseline grid, multiples for margins/padding.
- **Components**: Buttons, cards, inputs, modals – defined with CSS variables for easy theming (light/dark mode).

## Interaction
- Subtle micro‑animations on hover/focus.
- Smooth page transitions using CSS `transition` and `opacity`.
- Accessible focus outlines and ARIA attributes.
